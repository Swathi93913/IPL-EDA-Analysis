# IPL Data Analysis (Exploratory Data Analysis)

This project performs Exploratory Data Analysis on the IPL dataset using Python.

## Dataset
The dataset contains IPL match data including teams, players, stadiums, and match results.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed

1. Most Winning IPL Teams
2. Top Run Scorers in IPL
3. Stadiums Hosting the Most Matches

## Insights
- Identified the teams with the highest number of wins.
- Found the top run scorers in IPL history.
- Analyzed stadium trends based on number of matches played.

## Visualization
Graphs were created using Matplotlib to visualize the results.

## Project Files
- matches.csv
